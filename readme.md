# 99Tech Code Challenge

Solutions to the 99tech code challenge. Each problem is self-contained
under `src/`. Stack and run instructions live in the per-problem README.

## Problem 2 - Currency Swap Form
React + TypeScript + Vite + Tailwind. Fetches live token prices from
the provided endpoint and computes swap output in real time.
[Source](src/problem2/) · [README](src/problem2/README.md) · [Live demo](https://99tech-currency-swap.pages.dev)

## Problem 4 - Sum to N
Three TypeScript implementations of `sum_to_n`, each with notes on
time and space complexity. [Source](src/problem4/index.ts)

## Problem 5 - CRUD Server
Express + TypeScript + Prisma + PostgreSQL, packaged with Docker
Compose. CRUD over a `Task` resource with filters, validation, and
integration tests. [Source](src/problem5/) · [README](src/problem5/README.md)

## Problem 6 - Scoreboard Architecture
Specification for a real-time top-10 scoreboard module. Covers data
model, API, WebSocket fan-out, anti-cheat, and scalability.
[Source](src/problem6/README.md)

## Running locally
Each problem is self-contained. See the README in its `src/problemN/`
folder for run instructions.
